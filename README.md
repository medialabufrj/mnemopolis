# mnemopolis
## memória, política e cidade nas imagens sobreviventes

Este tema de Wordpress adiciona a interface de upload de imagens no frontend do site.  
Usuários podem se registrar através de suas contas nas redes sociais.  

Requer os plugins:

* Social Connect
* Members

E também as configurações:

* novos usuários são colaboradores
* adicionar `upload_files` as permissões de colaboradores
